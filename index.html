[Space] : Shoot game

How to play

  • Shoot a bubble at the already set bubbles above.
  • Use the left and right arrows to set the direction in which to shoot.
  • Match three or more like colors to clear the bubbles.
  • When all of the bubbles have been cleared, the game is over!
